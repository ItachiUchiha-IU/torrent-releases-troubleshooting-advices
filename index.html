Troubleshooting & Advices:

  • if a file doesn't open on Windows, it's because you put the files under an excessively long path: update or install one of the media players I recommend, or use Ctrl+Q to open the files, or move or rename the files

Use one of the following media players (highly recommended):

  • MPC-BE (remember to select MPC Video Renderer during the installation otherwise u have to manually install it)
    • optional: install madVR to have more video renderers to be choosen from
    • select your renderers from the matching tabs in Options (press O to open):
      • in Video select MPC Video Renderer (or madVR)
      • in Audio select MPC Audio Renderer
      • in Subtitles, select VSFilter as subtitle renderer (optional)
    • to fix karaoke subtitles flickering: in Options -> Subtitles -> Rendering set Sub pictures to buffer to 0
    • to show the chapter markers in the seekbar: Options -> Player -> Interface -> Use chapters marker
    • to use the preview in the seekbar: Options -> Player -> Interface -> Use the preview in the search
    • in Options -> Player uncheck Show OSD (optional)
    • change the settings in Player and in Windows size to your liking
  • MPC-HC
    • it may be tricky to install at first for the average user (especially if u already have MPC-BE) but it doesnt require ulterior installations for the best features if u choose the proper package (the Full package is recommended)
      • to install both BE and HC easily:
        • download and install MPC-BE following the instructions above
        • download the Full package of MPC-HC and run the installation selecting Install MPC-HC as a secondary player (u can leave everything else on default)
    • choose the renderers from: Options -> Playback -> Output :
      • for Video, select MPC Video Renderer or madVR
      • for Subtitles, select VSFilter (optional, but it can be useful in some cases: when some fonts are not muxed into the files and if the karaoke subs aren't perfectly aligned)
    • change the settings in Player and in User interface to your liking
  • MPV
    • not the easiest media player to set up & use for the average consumer
    • an external guide to installing and configuring mpv

Set your preferred languages for audio and subtitles (highly recommended):

the player will open the files in the languages u prefer (both audio and subs) and will select the full subs or signs based on the audio language selected!
  • on MPC-BE, go to Options -> External Filters and click on Add Filter, then add the Lav Splitter filter and open it
    • if u dont have the Lav Splitter filter, u need to install the LAV Filters (during the installation u can choose to install the LAV Splitter only) or, if u are still in doubt about installing both MPC-BE and HC, u can install MPC-HC as secondary player as it contains the LAV Filters
  • on MPC-HC, go to Options -> Internal Filters and click on Splitter to edit the LAV Filters
  • [LAV SPLITTER CONFIGURATION]
    • in Subtitle Selection Mode choose Advanced and check Reselect Subtitles on every Audio Switch (with this setting u can simply press A during the play to change the audio track and the subtitles will change automatically accordingly to your preferences)
    • in Audio write your preferred languages for audio (examples: jpn,ita,eng, eng,jpn, ita,eng,jpn etc.)
    • in Subtitles write your preferred languages for subtitles manually or simply copy-paste (and/or edit) a pre-made string (strings made by me are just below!)
    • save the configuration\ [LAV SPLITTER CONFIGURATION - END]
  • on MPC-HC u can now close the player and eveything should be set correctly
  • on MPC-BE u need to go to Priority (under External Filters) and change everything u can from default to LAV Splitter, then save and close the player
  • if everything is set properly, the filters will work; if they dont, then there are only two options:
    • if they dont work on specific sources, it means that those sources have the subs set/named improperly: u can try remuxing them correctly or editing the string in LAV Splitter
    • if they dont work at all, it means u did something wrong while following this guide... congrats, u played yourself, now u need to repeat every passage in this guide and be careful to apply every change XD Rip

More Troubleshooting and Advices:

  • use A to change the audio track, S for subtitles (if u set up the preferred languages as explained above, they will automatically change accordingly to your preferences when changing the audio track)
  • use Page Down (and Page Up) to move between the chapters of the video (it goes to the following episode if there are no more chapters)
  • use Ctrl + Page Down (and Ctrl + Page Up) to change episode
  • as of today, MPC-BE has still a bug regarding the issue about opening the files that are over the windows path limit; I posted an issue about it on their git-hub but...... anyway, as a temporary solution, to open such files, do File -> Quick Open File (u can do it faster by using the Ctrl+Q shortcut)
