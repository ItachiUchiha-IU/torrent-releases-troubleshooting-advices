Track Auto-Selection (highly recommended):

Automatically load your preferred languages for both audio and subtitles. The player will distinguish between full dialogue and signs/songs based on your audio selection, even updating subtitles instantly when you switch audio tracks.

MPV Track Auto-Selection:

  • Advanced Track Auto-Selector (.lua script) (highly recommended):
    Download or save the .lua file at /MPV_Track_Auto-Selection and put it into your scripts folder inside your mpv installation path (e.g. C:\Program Files\mpv\portable_config\scripts).
  • Very Basic Track Auto-Selection (mpv.conf):
    (cannot update when switching audio tracks, cannot reliably distinguish between full dialogue and signs/songs)
    set your preferred languages inside the mpv.conf file (e.g. alang=ja,jpn,en,eng, slang=en,eng)
    or use an app manager: https://mpv.rocks/manager/

LAV Splitter (External Track Auto-Selector for MPC-BE/HC):

  • on MPC-BE, go to Options -> External Filters and click on Add Filter, then add the Lav Splitter filter and open it
    • if u dont have the Lav Splitter filter, u need to install the LAV Filters (during the installation u can choose to install the LAV Splitter only) or, if u are still in doubt about installing both MPC-BE and HC, u can install MPC-HC as secondary player as it contains the LAV Filters
  • on MPC-HC, go to Options -> Internal Filters and click on Splitter to edit the LAV Filters

  • [LAV SPLITTER CONFIGURATION]
    • in Subtitle Selection Mode choose Advanced and check Reselect Subtitles on every Audio Switch (with this setting u can simply press A during the play to change the audio track and the subtitles will change automatically accordingly to your preferences)
    • in Audio write your preferred languages for audio (examples: jpn,ita,eng, eng,jpn, ita,eng,jpn etc.)
    • in Subtitles write your preferred languages for subtitles manually or simply copy-paste (and/or edit) a pre-made string (strings made by me are under My strings for LAV Splitter)
    • save the configuration
    [LAV SPLITTER CONFIGURATION - END]

  • on MPC-HC u can now close the player and eveything should be set correctly
  • on MPC-BE u need to go to Priority (under External Filters) and change everything u can from default to LAV Splitter, then save and close the player
  • if everything is set properly, the filters will work; if they dont, then there are only two options:
    • if they dont work on specific sources, it means that those sources have the subs set/named improperly: u can try remuxing them correctly or editing the string in LAV Splitter
    • if they dont work at all, it means u did something wrong while following this guide... congrats, u played yourself, now u need to repeat every passage in this guide and be careful to apply every change XD Rip

More Troubleshooting and Advices:

  • Use A (or Shift + A) to cycle forward and backward through audio tracks. Use S (or Shift + S) for subtitles.
    (If you configured your preferred languages as explained above, the subtitles will automatically update to match your preferences whenever you switch audio tracks.)
  • Use Page Down (or Page Up) to move between chapters. If no more chapters are available, the player will skip to the next (or previous) file in the folder.
  • Use Ctrl + Page Down (or Ctrl + Page Up) to jump directly to the next or previous episode.
  • MPC-BE Path Limit: MPC-BE currently has a bug that prevents it from opening files that exceed the Windows path length limit. As a workaround, use Ctrl + Q (shortcut for File -> Quick Open File) to open these files directly.
  • 10-bit Content on 8-bit Monitors: If colors appear overly saturated or "broken" when playing 10-bit content on an 8-bit monitor, try changing your video renderer (avoid the internal/system renderer) or, if using MPC-BE, apply these settings.
